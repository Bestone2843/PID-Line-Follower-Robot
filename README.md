# PID Line Follower Robot

## Overview
An Arduino-based line follower robot implementing PID control for stable path tracking.

## Hardware Used
- Arduino Uno
- IR Sensor Array
- DRV8833 Motor Driver
- DC Motors
- Li-ion Battery

## Control Strategy
Weighted sensor-based error calculation.
PID controller used to reduce oscillation and improve corner stability.

## PID Tuning
Kp = X
Ki = Y
Kd = Z

Tuned manually using trial-and-error and error response analysis.

## Results
- Reduced oscillation by __%
- Achieved stable turning at higher speeds
